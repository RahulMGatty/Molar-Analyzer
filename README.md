# Molar Analyzer

Custom 3D Slicer extension for mandibular molar and canal analysis using CBCT scans.

## Features

- Manual landmark placement
- Molar root localization
- Mandibular canal distance measurement

## Installation

1. Clone this repository
2. Open 3D Slicer
3. Developer Tools → Extension Wizard → Load Extension
